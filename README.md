# 🌐 Personal Portfolio Website

This is my personal portfolio website built to showcase my projects, technical skills, and development work.

## 🚀 Live Website
You can view the deployed portfolio here:

🔗 https://adityaverma19.netlify.app/

## 🛠️ Tech Stack

- React.js
- Tailwind CSS
- JavaScript
- HTML5
- CSS3

## 📂 Features

- Responsive modern UI
- Project showcase section
- Skills section
- Contact section
- Smooth navigation

## ⚙️ Deployment

The project is deployed using **Netlify**.

Steps used for deployment:
1. Build the project using `npm run build`
2. Upload the `dist` folder to Netlify
3. Configure build settings
4. Publish the site
